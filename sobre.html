Quem somos

Este site foi desenvolvido como parte de um trabalho acadêmico, utilizando apenas HTML e CSS. O objetivo é demonstrar a criação de páginas estáticas e a navegação entre elas sem uso de linguagens de programação dinâmicas.

Nossa missão

Compartilhar conhecimento sobre desenvolvimento web, incentivar boas práticas no uso de HTML e CSS, e mostrar como é possível criar páginas bem estruturadas mesmo com recursos básicos.

O que você encontra aqui

  • Página inicial com introdução ao tema
  • Pop-up de login/cadastro
  • Exemplo de navegação entre páginas
  • Conteúdo organizado em seções
